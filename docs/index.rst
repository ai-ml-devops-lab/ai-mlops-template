AI MLOps Template
=================

AI MLOps Template is a compact, production-style project skeleton for training,
versioning, documenting, and serving a machine learning model.

The repository demonstrates a complete local workflow:

* deterministic scikit-learn training;
* artifact, metrics, and model metadata generation;
* a FastAPI prediction service;
* automated tests, linting, and typing checks;
* Docker packaging;
* optional MLflow experiment tracking.
